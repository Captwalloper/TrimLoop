# Test Setup

## Automated

* Core functionality -- message based loop controls

### Automated Procedure

1. `npm run test` or open `test.html` in chrome
2. Check for 'All tests passed!'
3. Check for errors/warnings in console.

## Manual

* UI & keybinds

### Manual Procedure

1. Sideload extension
2. Reload extension
3. Reload test tab (ex. test.mp4)
4. Play with loop button
    * UI
    * Keybind
5. Play with loop bounds
    * UI
    * Keybind
6. Play with reset button
    * UI
    * Keybind
7. Reload popup
    * Also with keybind
    * Verify config was persisted

# The purpose of TrimLoop is:

1. Create a supported A-B looper for offline videos for the chrome browser that I like enough to use personally
2. Have a piece of software I've recently written to point at for portfolio reasons
3. Practice working with agentic AI

## Here's what I've decided so far about how it will function:

* similar to dark mode, the extension icon should show whether it's active or not
* ideally, pressing the icon should toggle on/off
* should be able to easily set/clear A and B, using the video's current time
* looping should be possible via UI

* probably keybinds, maybe mappable
